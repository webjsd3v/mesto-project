# Проект: «Место»

Цель проекта «Место» предоставить пользователю возможность создать мини-блог с картинками.

## Технологии, которые использовались

* Флексбокс верстка
* Адаптивная верстка
* Гриды
* БЭМ
* Кейфреймы
* Медиазапросы

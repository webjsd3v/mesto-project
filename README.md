# Проект: «Место»

Цель проекта «Место» предоставить пользователю возможность создать мини-блог с картинками.

## Технологии, которые использовались

* Флексбокс верстка
* Адаптивная верстка
* Гриды
* БЭМ
* Кейфреймы
* Медиазапросы
* Webpack
* PostCSS
* Babel

##Функциональность в проекте
1. Редактирование профиля
2. Добавление Карточки
3. Лайк
4. Удаление Карточки
